  • animation-count_X CLASS

    Special Class use, X = Repeat Number

    animation-count_3 = repeat animation 3 times.

    animation-count_infinite = loop animation infinity.

  • animation-duration_X CLASS

    Special Class use, X = Duration of animation in seconds.

    animation-duration_3 = Time animation 3 seconds.

  • base_fadeIn(element) FUNCTION

    element: Use QuerySelectorAll Logic ( .class, #id, div etc.. )

    Add class .fadeIn to element. If has class .fadeOut be removed

  • base_fadeOut(element) FUNCTION

    element: Use QuerySelectorAll Logic ( .class, #id, div etc.. )

    Add class .fadeOut to element. If has class .fadeIn whil be removed
